# 🚗 Smart Parking System
## 📌 Overview

The Smart Parking System is a web-based application designed to efficiently manage parking slots in real-time. It helps users find and book available parking spaces while enabling administrators to monitor and control parking operations.

---

## 🎯 Objectives

* Reduce time spent searching for parking
* Improve parking space utilization
* Provide a simple and user-friendly interface
* Demonstrate the use of algorithms in real-world applications

---

---

## 🧠 Algorithm Used

### Greedy Algorithm (Parking Slot Allocation)

The system uses a **Greedy approach** to allocate parking slots efficiently.

### 🔹 How it works:

* The system scans available parking slots
* Assigns the **first available slot** to the user
* Minimizes allocation time and ensures quick response

### 🔹 Why Greedy?

* Fast and efficient
* Suitable for real-time systems
* Reduces computational complexity
